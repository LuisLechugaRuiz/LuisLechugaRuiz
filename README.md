## Open-source work

### Aware SDK
[`aware-network/aware-sdk`](https://github.com/aware-network/aware-sdk)

The current public Aware release is **aware-sdk**, a CLI-first distribution for the Aware stack. It packages baseline infrastructure shared by the release pipeline and Studio-adjacent tooling, including:

- release and bundle tooling
- manifest-driven testing
- filesystem indexing / coordination
- environment contracts
- terminal runtime infrastructure

This is the current OSS rail for Aware. The broader Graph-OS direction will build on top of this foundation over time.

### Robotics / manipulation
#### general_manipulation
[`LuisLechugaRuiz/general_manipulation`](https://github.com/LuisLechugaRuiz/general_manipulation)
